Exact Solitaire package

Install by unzipping the release archive at the Kindle USB-storage root.
Launch from KUAL: Exact Solitaire -> Exact Solitaire.

Inspired by GNOME Aisleriot:
https://gitlab.gnome.org/GNOME/aisleriot

This package is a Kindle-friendly Windows Solitaire-style Klondike game, not
the full GNOME Aisleriot game collection. Draw 3 is the default classic
Klondike mode; use Draw 1 / Draw 3 to choose stock draw mode. Draw 3 displays
up to three waste cards and allows moving only the top/rightmost waste card.
